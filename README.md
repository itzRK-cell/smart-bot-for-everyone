# Smart Bot For Everyone

A Slack bot built with Node.js and Slack Bolt.

## Features

* `/smart-bot-for-everyone-ping` — Check bot latency
* `/smart-bot-for-everyone-help` — Show available commands
* `/smart-bot-for-everyone-catfact` — Get a random cat fact from an external API

## Technologies Used

* Node.js
* Slack Bolt
* Axios
* GitHub
